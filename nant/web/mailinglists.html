NAnt Mailing Lists

Please read the "Guide to SourceForge.net Mailing List Services" before subscribing and posting to any of the lists below.

All mailing lists are subscriber only lists, this means that you have to subscribe before you can post to the list. Avoid sending HTML mails to any of these lists.

Announce List

Low Traffic Subscribe Unsubscribe

This list is only for announcing new releases of NAnt, and related open-source or commercial products.

User List

High Traffic Subscribe Unsubscribe Archive

This list is for user related questions.  Having questions posted to this list may allow others having the same problem to solve it on their own.

Developer List

High Traffic Subscribe Unsubscribe Archive

This is the list where participating developers of the NAnt build tool meet and discuss issues, code changes/additions, etc.

Commits List

High Traffic Subscribe Unsubscribe Archive

Subscribers to this list get notices of each and every code change, build results, testing notices, etc.
