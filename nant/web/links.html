NAnt Links

Reports of broken links and suggestions are welcome and encouraged.

  • NAnt Home Page
    The official home page for NAnt. Useful if you are reading this offline.
  • NAntContrib Home Page
    The official home page for NAntContrib. This site contains useful non-core tasks and utilities for NAnt.
  • Draco.NET
    A continous build server written in C#. This build server runs as a Windows Service. Draco.NET works with NAnt, MSDevEnv, CVS, SourceSafe, and various other build environments and source repositories.
  • NUnit
    A simple framework to write repeatable tests in any .NET language.
  • NDoc
    Merges the assemblies and XML documentation files produced by csc.exe and creates a compiled HTML Help file enabling you to browse your own libraries just like the .NET Framework Class Library.
  • log4net
    A port of the excellent log4j framework to the .NET runtime.
  • NAnt Print Ready Logo
    A encapsulated postscript version of the NAnt logo, perfect for printing.
  • Jakarta Ant Home Page
    The project that started the idea and the definitive build tool for Java applications.
  • Portable.NET's Csant
    Csant is an XML-based build tool, in the style of nant, that is written in C. This makes it ideal for bootstrapping C# libraries and tools that would otherwise require manual intervention to bootstrap.
  • Artistic Style
    A Free, Fast and Small Automatic Indentation Filter for C# and other source codes.
  • SharpCvsLib
    A cvs client library written in c#.
