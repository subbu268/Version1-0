NAnt External Tools and Extensions

This page lists external resources for NAnt: Tools, Extensions. Reports of broken links and suggestions are welcome and encouraged.

Tools

  • ColorNAnt

     The Perl script ColorNAnt is a wrapper for 'NAnt' and produces the same output but with pretty 'syntax' highlighting. Colour schemes can be customized.
    Compatibility:All versions
    URL:http://colornant.googlepages.com
    Contact:aritmeester+colornant@gmail.com
    License:GNU General Public License

  • NAntBuilder

     NAntBuilder is an NAnt IDE with code completion, syntax highlighting and script debugging. It is designed to be a powerul NAnt script creator, editor, and debugger.
    Compatibility:NAnt 0.91 Alpha2
    URL:http://www.polestarsoft.com/
    Contact:support@polestarsoft.com
    License:Commercial

  • NAntGraph

     NAntGraph is a documentation tool designed to make it easier to understand NAnt or Ant build files by producing a diagram showing targets and dependencies.
    Compatibility:NAnt 0.85
    URL:http://www.nichesoftware.co.nz/nantgraph
    Contact:bevan@nichesoftware.co.nz
    License:GNU General Public License

  • NAntPad

     NAntPad is a concise tool for developing and editing build files for NAnt. The UI is similar to explorer with a tree navigator, and an editing and viewing pane for tasks.
    Compatibility:NAnt 0.85
    URL:N.A.
    Contact:N.A.
    License:Commercial

  • NAnt Wizard

     NAnt Wizard is a compact and lightweight GUI front-end for NAnt. Application is contained in a single small exe file that is why it can be put into your version control system along with source code.
    Compatibility:NAnt 0.85
    URL:http://www.nantwizard.com
    Contact:Contact Page
    License:Free for non-commercial use

Extensions

  • NAntContrib

     NAntContrib is the project for tasks and tools that haven't made it into the main NAnt distribution yet or for whatever reason don't belong there.
    Compatibility:NAnt 0.85
    URL:http://nantcontrib.sourceforge.net
    Contact:Developer mailing list
    License:GNU General Public License

  • NAntScript

     NAntScript provides custom tasks that allow you to script other custom tasks using regular NAnt script.
    Compatibility:NAnt 0.85
    URL:http://code.google.com/p/nantscript/wiki/NAntScript
    Contact:flukefan@googlemail.com or neil.paterson.logan@googlemail.com
    License:GNU General Public License

  • StyleCopCmd

     StyleCopCmd provides both a command-line interface and NAnt task for Microsoft's source code analysis tool, StyleCop.
    More than just a simple front-end, StyleCopCmd produces beautiful HTML reports about any violations it discovers.
    Compatibility:NAnt 0.86
    URL:http://stylecopcmd.sourceforge.net
    Contact:sakutz@gmail.com
    License:BSD License
